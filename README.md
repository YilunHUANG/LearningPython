# LearningPython

It's the python learning note of MIT's open course named Introduction to Computer Science and Programing

PyDev: http://pydev.org/updates

注意一个问题：  
（10/22更新）  
目前大部分的包都只更新到3.4，所以安装的话还是安装3.4  
另外，3.4版本有32位和64位，有些包只支持32位，所以还是装32位

各个包的作用  
NumPy：各种科学计算，  
SciPy：实现matlab的功能  
Matplotlib：绘制图形
